# TV-DB search app

## Setup

1. Clone this repo and cd into the root folder.
2. run 
  ```bash
  yarn install
  ```
3. start
  ```bash
    yarn start
  ```
4. Go to http://localhost:3000/

This app was created using create-react-app.
